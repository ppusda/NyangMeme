# 냥밈 (NyangMeme) 🐱

재미있는 고양이 밈(Meme)들을 모아놓은 웹 애플리케이션입니다.

🔗 **배포 주소**: [https://nyangmeme.bbgk.me](https://nyangmeme.bbgk.me)

## 🌟 주요 기능

- 다양한 고양이 밈들을 구경할 수 있습니다.
- 밈들에 대한 설명과 밈이 어떻게 활용되고 있는지 확인할 수 있습니다.

## 🛠️ 기술 스택

### Backend
- Java
- Spring Boot

### Frontend
- JavaScript
- React
- Vite
- Tailwind CSS
